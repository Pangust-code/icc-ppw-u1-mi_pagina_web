Practica - Estandares Web

Estos estandares son HTML, CSS y JS, los cuales son las bases fundamentales de cualquier pagina web moderna. Estos cumplen una función específica y permiten que los sitios tengan estructura, sea atractivo y funcional.

Lenguaje Descripción
HTML (HyperText Markup Language) Ayuda a definir el contenido y la estructura de la pagina web Es el esqueleto de la pagina, usa etiquetas (<p>, <h1>, <a>) y hace que los navegadores comprendan la información.
CSS (Cascading Style Sheets) Da estilo y forma visual al contenido dado de HTML. Controla colores, tipografias, margenes, bordes, posiciones, etc. Ayuda a adaptar el diseño a distintos dispositivos.
JS (Java Script) Sirve para añadir dinanimos y funcionalidad a la pagina. Permite interactividad con la pagina, controlas el contenido y los estilos en tiempo real (DOM), por ultimo se usa tambien para frameworks modernos.
